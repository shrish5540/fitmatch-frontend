# 🏋️ FitMatch

FitMatch is an AI-powered gym partner finder that helps fitness enthusiasts connect with like-minded workout partners.

## 🚀 Live Demo

https://fitmatch-frontend-alpha.vercel.app

## ✨ Features

* 🔐 JWT Authentication
* 👤 Profile Management
* 📸 Profile Photo Uploads (Cloudinary)
* ❤️ Swipe-Based Matching
* 🤝 Match Creation
* 💬 Real-Time Chat
* 🤖 AI Fitness Coach
* 🏋️ AI Workout Generator
* 🥗 AI Diet Generator
* 🔥 Daily Streak Tracking
* 📈 Progress Dashboard
* 🏆 Leaderboard System

## 🛠 Tech Stack

### Frontend

* React
* Vite
* Tailwind CSS
* Axios
* React Router

### Backend

* Node.js
* Express.js
* MongoDB Atlas
* JWT Authentication
* Cloudinary
* Groq AI

## 🌐 Deployment

* Frontend: Vercel
* Backend: Render
* Database: MongoDB Atlas
